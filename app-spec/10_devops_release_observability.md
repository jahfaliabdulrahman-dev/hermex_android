# 10 — DevOps, Release & Observability

## Build
- `flutter build apk --release` for Android
- `flutter build ipa` for iOS

## CI/CD (future)
- GitHub Actions for automated builds
- Version bump via semantic-release

## Release Gates
- [ ] All tests pass
- [ ] Zero-trust audit clean
- [ ] APK size < 30MB
- [ ] API server connectivity verified

## Versioning
