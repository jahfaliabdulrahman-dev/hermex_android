# 03 — User Flows & Navigation

## Primary Flows
1. **First Connection** → Enter server URL + API key → Health check → Chat
2. **Chat** → Type → Send → SSE streaming → Tool progress visible
3. **Sessions** → Browse → Search → View messages → Manage
4. **Tasks** → List cron jobs → View details → Edit/Delete/Pause/Run

## Navigation
Bottom Navigation: Chat | Sessions | Tasks | Workspace | Settings
Settings sub-screens: Skills, Memory, Insights

## States
Every screen: Loading, Empty, Error, Success, Offline
