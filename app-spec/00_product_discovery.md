# 00 — Product Discovery

## Problem
Self-hosted Hermes Agent users need mobile access. Android users have NO native app — only iOS (Hermex, via hermes-webui middleware) or web dashboard. Hermex Android fills this gap with direct API Server connection.

## Solution
Flutter-based mobile app connecting directly to Hermes Agent API Server. No middleware needed.

## Market
Hermes Agent community (15K+ stars). All Android-using self-hosters.

## Success Metrics
- APK installs & works first try
- Chat SSE streaming < 500ms latency
- All 8 MVP features functional
